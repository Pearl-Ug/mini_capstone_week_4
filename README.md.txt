README file and team contributions
The project shows a quick summary of the scores on three main KPIs for a hospital, namely, average wait time, daily occupancy, and daily scan volume, as a multi-panel dashboard.

How to run the project
1.	Clone the repository
2.	Open the project folder in any python environment (VS code, Jupyter or Colab)
3.	Install required libraries
4.	Run all cells in order from top to bottom
5.	Follow the on-screen prompts
6.	View output dashboard
